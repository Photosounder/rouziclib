= rouziclib documentation
:toc:
:toclevels: 2
:source-highlighter: highlight.js

This directory documents the current rouziclib source tree: a portable C utility and application-support library covering numeric types, geometry, raster graphics, text, GUI controls, file I/O, audio, threading, optional compute backends, and a small expression interpreter.

The public umbrella files are link:../rouziclib.h[rouziclib.h] and link:../rouziclib.c[rouziclib.c]. The headers in those files are the authoritative API surface; the thematic guides here explain how the pieces fit together and point to the source headers for exact declarations.

== How to use these docs

Examples use C99-style declarations and the names that are present in the current headers. They are deliberately small: include the relevant module through `rouziclib.h`, then consult the linked header for the complete parameter list and ownership rules.

The Spacewar references are examples of usage, not a separate compatibility layer. They are especially useful for seeing how the library's global framebuffer, mouse, keyboard, zoom, font, and window-manager state is assembled into a real application.

== Source of truth

When this documentation and a header disagree, trust the header and update the relevant `.adoc` file. `quick_reference.h` remains a useful cookbook of short API examples, while the module headers contain the complete declarations and compile-time guards.
